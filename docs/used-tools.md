# TypeScript

[Why TypeScript?](https://www.typescriptlang.org)

The current `webpack-config` enables the use of TypeScript. However, in React projects it's not required, you are free to use vanilla JavaScript if you prefer. For your editor to support you during your work, you might want to consider adding JSDoc comments to your code.
